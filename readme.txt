 == Installation ==

1. Upload the `Bank-Parsian-EDD-gateway`  files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Plugin Name screen to configure the plugin
4. In the payement gateways setting of the EDD plugin put the payement gateways and default gateways on "Bank Parsian"
   complete the PIN  field and click save changes.
